1. Power Shell: Instalar ffmpeg desde
winget install ffmpeg

2. CMD: En la carpeta del proyecto ejecutar estos comandos de instalacion
venv\Scripts\activate
pip install -r requirements.txt

3. CMD: Comandos para ejecucion
venv\Scripts\activate
fastapi src/main.py
start http://127.0.0.1:8000/
